## モチベーショングラフ

### ユーザージャーニーと感情曲線

```
感情レベル
    ^
 高 |                                    ⭐ 予測結果を見て
    |                                   /   「これは売れる！」と確信
    |                              ____/
    |                             /
 中 |        商品説明を         /
    |        入力する          /
    |    ___/\________________/
    |   /    \
    |  /      シミュレーション
 低 | /       実行中（不安）
    |/
    +-------------------------------------------------> 時間
      認知  検討  利用開始  シミュレーション  結果確認  改善
```

---

## 各フェーズの詳細

### Phase 1: 認知（Awareness）

| 項目 | 内容 |
|------|------|
| ユーザーの状態 | 新商品を出したいが、売れるか不安 |
| 感情 | 不安、期待 |
| 行動 | 「レビュー予測」「商品評価シミュレーション」で検索 |
| モチベーション | 失敗リスクを減らしたい |

### Phase 2: 検討（Consideration）

| 項目 | 内容 |
|------|------|
| ユーザーの状態 | Review Predictorを発見 |
| 感情 | 興味、懐疑 |
| 行動 | 機能説明を読む、デモを試す |
| モチベーション | 本当に役立つか確認したい |

### Phase 3: 利用開始（Onboarding）

| 項目 | 内容 |
|------|------|
| ユーザーの状態 | 商品説明文を入力 |
| 感情 | 期待、少しの不安 |
| 行動 | 説明文を入力してシミュレーション実行 |
| モチベーション | 自分の商品がどう評価されるか知りたい |

### Phase 4: シミュレーション実行（Processing）

| 項目 | 内容 |
|------|------|
| ユーザーの状態 | 約5秒間の待機 |
| 感情 | 緊張、期待 |
| 行動 | ローディング画面を見守る |
| モチベーション | 早く結果を見たい |

### Phase 5: 結果確認（Result）

| 項目 | 内容 |
|------|------|
| ユーザーの状態 | 予測結果を確認 |
| 感情 | 満足 or 改善意欲 |
| 行動 | 評価分布、セグメント分析、サンプルレビューを確認 |
| モチベーション | 具体的な改善ポイントを見つけたい |

### Phase 6: 改善（Improvement）

| 項目 | 内容 |
|------|------|
| ユーザーの状態 | 説明文を修正して再シミュレーション |
| 感情 | 前向き、挑戦 |
| 行動 | 説明文を改善、再度シミュレーション |
| モチベーション | より高い評価を得たい |

---

## ユーザーの主要モチベーション

### 1. リスク回避

```
「売れない商品を出して在庫を抱えたくない」
```

- 新商品の失敗リスクを事前に把握
- データに基づいた意思決定

### 2. 時間短縮

```
「A/Bテストに何週間もかけたくない」
```

- 実際に売り出す前に予測可能
- 複数パターンを素早く比較

### 3. 根拠の提示

```
「上司に説明できる客観的なデータがほしい」
```

- 10,000人のシミュレーション結果
- セグメント別の詳細分析

### 4. 改善の方向性

```
「何をどう直せばいいかわからない」
```

- セグメント別の評価傾向
- サンプルレビューによる具体的フィードバック

## ROI（投資対効果）

### 定量的メリット

| 項目 | Before | After | 改善率 |
|------|--------|-------|--------|
| 商品説明作成時間 | 2時間 | 30分 | **75%削減** |
| A/Bテスト期間 | 2週間 | 5分 | **99%削減** |
| 新商品成功率 | 30% | 60% | **2倍** |

### 定性的メリット

- 商品企画への自信向上
- データドリブンな意思決定文化の醸成
- 顧客視点でのマーケティング強化
